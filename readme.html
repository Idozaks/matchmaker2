Mycelium Matchmaker DIU Setup

Follow along this guide to set up your Digital Identity Unit (DIU) with AI-powered personalization.

Step 1: Overview

This multi-step setup wizard helps you create a personalized digital profile. The wizard consists of 5 steps:

  1. Welcome: Provide your name, optional pronouns, and set profile visibility.
  2. Interests: Choose your interests from a preset list.
  3. Content Preferences: Select the types of content you prefer (Articles, Videos, Podcasts, etc.).
  4. Privacy Settings: Toggle AI personalization for privacy management.
  5. Complete: Review your DIU profile summary and explore your AI-powered feed.

Step 2: Features

  • Responsive Design: Adapts seamlessly on mobile screens.
  • Real-Time Interactivity: Powered by Alpine.js for dynamic updates.
  • Step-by-Step Navigation: Navigate forward and backward through the wizard.

Step 3: Getting Started

To try out the DIU setup wizard:

  1. Clone the repository or download the file.
  2. Open matchmaker2.html in your browser.

The project uses CDN links for both Tailwind CSS and Alpine.js, so no additional setup is needed.

Step 4: File Structure & Technologies

File Structure

  • matchmaker2.html: Contains the HTML markup and inline Alpine.js scripts.

Technologies Used

  • HTML5: Provides the structure.
  • Tailwind CSS: Styles and makes the layout responsive.
  • Alpine.js: Adds lightweight reactivity and interactivity.

Step 5: Usage Instructions

  1. Start the Wizard:

    Open matchmaker2.html in a browser.

  2. Fill in Your Profile Information:
    • Enter your name and optional pronouns.
    • Select your profile visibility.
    • Choose your interests and content preferences in the dedicated steps.
    • Toggle the AI personalization for privacy controls.
  3. Complete the Setup:

    Review your DIU profile summary and click the button to explore your personalized AI-powered feed.

Enjoy the experience and feel free to further customize the project to suit your needs!
